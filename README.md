## 🚀 Installation

### Prerequisites
- Python 3.9+
- MySQL Server

```bash
git clone https://github.com/yourusername/premium-subscription-manager.git
cd premium-subscription-manager
pip install -r requirements.txt
```

## 📖 Usage

1. Configure database in `config.py`
2. Run the application:
```bash
python app.py
```
